# VocabHelper Support Page

Static support page for VocabHelper AI, intended for GitHub Pages and App Store Connect.

## Contents

- `index.html` contains the full support page.
- `.nojekyll` ensures GitHub Pages serves the site as plain static files.

The page currently includes:

- support contact email
- App Store review friendly support copy
- expanded FAQ section with collapsible answers

## Publish To GitHub Pages

1. Create or use a public repository for the support site, for example `vocabhelper-support`.
2. Push the contents of this folder to that repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main` and folder `/ (root)`.
6. Save and wait for the deployment to finish.

Expected default Pages URL:

`https://<github-username>.github.io/vocabhelper-support/`
