# VocabHelper Support Site

Minimal static support page for GitHub Pages.

## Publish

1. Create a new public GitHub repository, for example `vocabhelper-support`.
2. Push the contents of this folder as the root of that repository.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main` and folder `/ (root)`.
6. Save and wait for Pages to publish.

Expected URL:

`https://<github-username>.github.io/vocabhelper-support/`

Use that URL in App Store Connect as the Support URL.
