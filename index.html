FAQ

How do I add new words?

Open a category in the app, then use the add button to create a new word with translation, examples, and optional notes.

Can I import or export my vocabulary?

Yes. VocabHelper supports CSV import and export so you can move vocabulary in or out of the app.

Do I need an account?

You can use core features locally, but signing in is required for cloud sync and some account-based functionality.

What are AI credits used for?

AI credits are used for AI-powered features such as generated words and some listening or language assistance workflows.

What should I include in a support email?

Include your device model, OS version, app version, and a short description of the issue so we can reproduce it faster.
